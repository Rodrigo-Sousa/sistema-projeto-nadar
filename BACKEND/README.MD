# BackEnd da aplicação

Esta parte é a responsável por todo o _backend_ da aplicação, contendo as regras de negócio e realizando a interface entre o _frontend_ e o _Banco de dados_.

- Escalável
- Organizado (MVC + Camadas)
- Seguro
- Preparado para integração com o frontend

---

## _Stacks_ utilizadas:

- **Node.js + Express**
- **MySQL**
- **Sequelize**
- **JWT** para autenticação
- **dotenv** para variáveis de ambientes
- **bcrypt** para senha.


## Rodando o projeto

Assim que baixar, abrir o projeto no VsCode (ou no editor de sua preferência) e no terminal, rodar o comando: `yarn install`, para que seja instalado todas as suas dependências e possamos utilizar o projeto.

### As dependências principais do projeto: 

- Express
- Mysql2
- Sequelize
- Dotenv
- Cors
- Bcrypt 
- JsonWebToken

### Dependência de desenvolvimento:

- nodemon
